# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.5.x   | :white_check_mark: |
| < 0.5   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue.
2. Email **boxiyu999@proton.me** with details including:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
3. You will receive a response within 72 hours.
4. A fix will be developed privately and released as a patch.

## Scope

This project provides security guardrails for LLM applications. Vulnerabilities in the guardrail detection logic (bypass techniques, false negatives) are in scope and welcome.
